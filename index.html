Introduction

This is a plugin for Apple Aperture to export to Gallery 3. The current release has plenty of bugs for sure, and very minimal error checking - but it works at least on my system.

Please consider making a donation.

If you have bugs or comments, please use the ApertureToGallery3git hub home page to connect with the author and user community.

Instructions

Most of the functionality is very obvious, but there are some places where additional explaination is useful:

  • Adding a Gallery - When adding a gallery, use the full url such as http://www.server.com/gallery3. Don't include the "index.php" portion of the url. When adding a gallery, the plugin needs a name - which could be anything useful for you, the url, the login name, and the API key. The password is only required to automatically retrieve the API key and will not be saved.
  • Watermarks - As of version 0.65.07, the ability to automatically apply a watermark is included. The resulting file will be a jpeg. The watermark image will be blended using a SoftLightBlendFilter. In theory, 50% gray will not cause the underlying image to be dodged or burned. Experimentally, I found that creating a 50% gray background with some text did quit a bit of dodgeing; I found that 27% gray worked better. This may have to do with the changing color space.

Requirements

This package takes at least OS 10.6 and requires Apple Aperture.
